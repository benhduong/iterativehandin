UI/UX: Iterative Design of Emerge Career

Original Website

As a surfer who is constantly breaking his boards, I want to redesign my local board repair guy's website to improve user experience.

Redesigned Website

Sketching and Wireframing

Current issues include:

  • Usabilty
    • Harsh colors of the website make reading unappealing.
  • Learnability
    • Website is very intuitive and information is plainly available, thus the website is very easy to learn. I want to keep this aspect in the redesign.
  • Memorability
    • The website is very memorable as the information is simple; however, I wish to rearrange the layout to improve memorability.

Accessibility

The WAVE report of the website determined that there was a missing alternative text for an image and a missing form label. These missing labels and alternative texts may cause confusion for users relying on screen readers to navigate through websites. Thus it is important that I fix these issues when redesigning the website. Although there were no contrast issues, the colors used are harsh to the eyes and make scrolling through the website unappealing, another issue I wish to fix.

Takeaways

Overall, the redesign improved the layout of the original website, by improving the order of how a user would interact with the information. I believe that a user would want to know what the service is, how much it consts, then find out the contact information. Additionally, I changed the color palatte to be easier on the eyes while still preserving the original theme of the website. With these changes, I believe that the website improves its usability, learnability, and memorability.
